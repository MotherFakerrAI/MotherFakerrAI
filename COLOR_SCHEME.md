# MotherFaker Studio 品牌配色方案

## 主色调

| 颜色 | 色值 | 用途 |
|------|------|------|
| **品牌紫** | `#BF00FF` | 主标题、进度条、强调色 |
| **品牌橙** | `#FF6600` | 次要强调、按钮、链接 |
| **科技蓝** | `#00A8FF` | 终端提示符、链接悬停 |

## 背景色

| 颜色 | 色值 | 用途 |
|------|------|------|
| **深空黑** | `#0D1117` | 主背景（GitHub 暗色） |
| **碳素黑** | `#1C2128` | 卡片背景、进度条底 |
| **纯白** | `#FFFFFF` | 主要文字 |

## 文字色

| 颜色 | 色值 | 用途 |
|------|------|------|
| **主文字** | `#FFFFFF` | 标题、正文 |
| **次文字** | `#8B949E` | 描述、次要信息 |
| **强调紫** | `#BF00FF` | 百分比、重点 |
| **强调橙** | `#FF6600` | 百分比、重点 |

## 终端风格配色

```bash
# 终端提示符
motherfaker@studio:~$     # 青色 #00A8FF

# 命令输出
> 内容                    # 白色 #FFFFFF

# 进度条字符
████████                  # 紫色渐变 #BF00FF → #8B00FF
░░░░░░                    # 灰色 #1C2128
```

## 使用示例

### 进度条
- Python/TypeScript 等技术 → 紫色渐变
- JavaScript/Node.js 等 → 橙色渐变

### 徽章
- 主要项目 → 紫色背景
- 工具/框架 → 蓝色/黑色背景

### 文字
- 标题 → 白色 + 紫色阴影
- 链接 → 橙色悬停变紫色

## SVG 渐变定义

```xml
<defs>
  <!-- 紫色渐变 -->
  <linearGradient id="purpleGrad" x1="0%" y1="0%" x2="100%" y2="0%">
    <stop offset="0%" style="stop-color:#BF00FF" />
    <stop offset="100%" style="stop-color:#8B00FF" />
  </linearGradient>
  
  <!-- 橙色渐变 -->
  <linearGradient id="orangeGrad" x1="0%" y1="0%" x2="100%" y2="0%">
    <stop offset="0%" style="stop-color:#FF6600" />
    <stop offset="100%" style="stop-color:#FF8800" />
  </linearGradient>
</defs>
```

## 响应式考虑

- SVG 进度条：最小宽度 200px
- 项目卡片：移动端单列显示
- 文字大小：最小 12px

---

**设计原则：**
- 科技感 + 品牌识别度
- 暗色主题护眼
- 高对比度易读
- 渐进式动画增强体验
